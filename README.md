# Notes
## Основные возможности
*   **Управление заметками**: Создание, редактирование и удаление текстовых заметок.
*   **Система тегов**: Добавляйте теги к заметкам для лучшей организации и быстрой фильтрации.
*   **Поиск**: Быстрый поиск заметок по заголовку прямо на главном экране.
*   **Фильтрация**: Просмотр заметок по тегам.
*   **Черновики**: Автоматическое сохранение состояния при выходе из экрана редактирования.
*   **Локальное хранение**: Все ваши данные хранятся локально на устройстве и доступны без интернета.

## Стек технологий и аргументация
*   **Room Persistence Library**: Используется для надежного локального хранения данных. Позволяет работать с базой данных SQLite через типизированные объекты и поддерживает реактивные потоки данных.
*   **MVVM (Model-View-ViewModel)**: Архитектурный паттерн, обеспечивающий четкое разделение бизнес-логики и пользовательского интерфейса, что упрощает тестирование и поддержку кода.
*   **Kotlin Coroutines & Flow**: Для выполнения асинхронных операций (работа с БД) без блокировки основного потока.
*   **KSP (Kotlin Symbol Processing)**: Для быстрой компиляции аннотаций Room.

## Инструкция по сборке

Для сборки и запуска приложения вам понадобятся:
*   **Android Studio** (рекомендуется версия Ladybug 2024.2.1 или новее).
*   **JDK 17** (установлен по умолчанию в современных версиях Android Studio).
*   Устройство или эмулятор с **Android 12.0 (API 31)** или выше.

### Шаги:

1.  **Клонируйте репозиторий**:
    ```bash
    git clone https://github.com/your-username/your-repo-name.git
    ```
2.  **Откройте проект**: Запустите Android Studio и выберите `Open`, затем укажите путь к папке с проектом.
3.  **Синхронизация**: Дождитесь завершения синхронизации Gradle-проекта.
4.  **Запуск**: Нажмите кнопку `Run` (зеленый треугольник) на верхней панели инструментов, предварительно выбрав целевое устройство или эмулятор.
